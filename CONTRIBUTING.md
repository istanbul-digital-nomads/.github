# Contributing to Istanbul Digital Nomads

Thanks for your interest in contributing! This guide applies to all repos in the organization.

## Ways to Contribute

- **Share knowledge** — Open a PR to add or update content
- **Report issues** — Found a bug or outdated info? Open an issue
- **Suggest ideas** — Feature requests and improvements are welcome
- **Help others** — Answer questions and review PRs

## How to Submit Changes

1. Fork the repository
2. Create a branch (`feature/your-change` or `fix/your-fix`)
3. Make your changes
4. Submit a pull request

## Content Guidelines

- Write in clear, accessible English
- Be specific — include addresses, links, and prices where relevant
- Date your contributions so readers know how current the info is
- Label any promotional content honestly

## Code of Conduct

All contributors must follow our [Code of Conduct](https://github.com/istanbul-digital-nomads/community/blob/main/CODE_OF_CONDUCT.md).

## Questions?

Open an issue or reach out in our [Telegram group](https://t.me/istanbul_digital_nomads).
