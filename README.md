# Istanbul Digital Nomads - Organization Defaults

This repository holds organization-wide community health files for [istanbul-digital-nomads](https://github.com/istanbul-digital-nomads). GitHub automatically applies these to all org repos that don't define their own.

## What's Here

| File | Purpose |
|------|---------|
| [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) | Community standards (Contributor Covenant 2.1) |
| [CONTRIBUTING.md](CONTRIBUTING.md) | How to contribute to any repo |
| [SECURITY.md](SECURITY.md) | Vulnerability reporting guidelines |
| [FUNDING.yml](FUNDING.yml) | Sponsorship links |
| [PULL_REQUEST_TEMPLATE.md](PULL_REQUEST_TEMPLATE.md) | Default PR checklist |
| [ISSUE_TEMPLATE/](ISSUE_TEMPLATE/) | Default issue forms |

## How Inheritance Works

These files are used as defaults. If a repo has its own version of any file, the repo-specific version takes priority.

For example:
- The `community` repo has its own `CONTRIBUTING.md` (the detailed, canonical version)
- The `website` repo has its own issue templates and PR template (tailored for code)
- All other repos inherit the defaults from here
